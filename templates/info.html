FAQ

What is this site?

EasySearch is a modern, open-source directory for Harvard courses and instructors. Most online course directories can be tedious and confusing to use. This site aims to provide a better user experience for this information.

Where does this site get its data?

This site gets all data from the Harvard Open Data Project. Spreadsheets have been manually downloaded and loaded into the database as two main objects: Instructors and Courses.

When was this created?

I started programming this site on September 26th, 2023. This means that until I automate this process, I manually upload each new semester's "Course Enrollment Stats" dataset from the HODP website and might possibly forget in the future.

How far does this information go back?

This information goes back 6 semesters, until Spring of 2021.
